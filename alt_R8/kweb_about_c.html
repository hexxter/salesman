Keyboard Commands
By default, all keyboard commands are enabled. They have to be used together with the left ALT-Key, except in kiosk mode, when the ALT option is not set. The full set of commands is:

URL for Start Page and (Optionally) Home Page
If nothing is entered here, kweb's menu page will be the start page and home page (called from the 'Home' button). You can also create your own "homepage.html" file in /usr/local/share/kweb and then this will be used automatically.
The new configuration will only take effect after restarting Minimal Kiosk Browser.
